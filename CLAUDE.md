# CLAUDE.md

## Objective
Run deterministic code-edit sessions across weak, free-tier, and local models.

## Work Protocol
1. Read relevant files first.
2. Plan in explicit numbered steps.
3. Execute one scoped step at a time.
4. Validate after each change.
5. Keep responses short and concrete.

## Safety
- Never read secrets or key material.
- Never run destructive shell commands.
- Never invent files or paths.
- Never perform broad rewrites without need.
